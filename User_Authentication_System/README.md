# User Authentication System

A simple Django authentication demo.

## Features

- User registration
- User login
- User logout
- Protected dashboard
- Modern dark UI

## Structure

```
accounts/          # Authentication app
templates/         # HTML templates
static/           # CSS and JS
db.sqlite3        # Database
manage.py         # Django CLI
```

## URLs

- `/` - Home
- `/accounts/signup/` - Sign up
- `/accounts/login/` - Login
- `/accounts/logout/` - Logout
- `/dashboard/` - Dashboard (protected)
- `/admin/` - Admin panel

## Tech

- Django 5.x
- SQLite
- Custom CSS
